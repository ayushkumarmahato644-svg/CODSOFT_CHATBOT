Project Details
Project Title: Rule-Based Chatbot (CorporateBot) Using Python

Objective:

To develop an automated virtual assistant designed to handle basic corporate inquiries.

To understand text processing and intent recognition using regular expressions (re).

To implement structured, rule-based natural language processing (NLP) logic.

To build an interactive, resilient command-line interface (CLI) for real-time user communication.

Key Activities
Text Preprocessing: Clean and normalize user inputs by stripping whitespace and converting text to lowercase for reliable matching.

Intent Recognition: Use regular expressions with word boundary markers (\b) to match user phrases (greetings, identity, capabilities, and farewells) without false positives.

Fallback Management: Implement a catch-all safety response (else clause) to elegantly handle unrecognized queries and ask for clarification.

Interactive Loop: Build a continuous processing cycle using a while True loop to keep the chat active dynamically.

Graceful Termination: Check for termination keywords (exit, quit, bye) to break the loop safely, and use exception handling (KeyboardInterrupt, EOFError) to manage unexpected terminal closures.

Technologies Used
Programming Language: Python

Core Libraries: re (Regular Expressions), sys (System-specific functions)

Technique: Rule-Based Pattern Matching (Natural Language Processing)

Concepts: Flow Control (Conditional Matrix, Infinite Loops), Input Sanitization, String Methods, Error and Exception Handling

Development Environment: Visual Studio Code, PyCharm, IDLE, or any standard Python IDE

Operating System: Cross-platform (Windows, Linux, macOS)
